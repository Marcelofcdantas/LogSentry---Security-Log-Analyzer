# LogSentry – Security Log Analyzer

LogSentry is a backend tool designed to analyze server logs and detect suspicious activity patterns such as brute-force login attempts, abnormal request rates, and repeated access to sensitive endpoints.

The project demonstrates backend development, log parsing, and basic cybersecurity analysis techniques by automatically processing log files and generating structured reports.

---

## Overview

Modern applications and servers generate large volumes of logs. Identifying suspicious behavior manually can be time-consuming and error-prone.

LogSentry automates this process by parsing log files and applying detection rules to identify patterns that may indicate security issues.

Examples of suspicious activity detected include:

- Repeated failed login attempts (possible brute-force attacks)
- Multiple requests to sensitive endpoints
- Unusual request frequency from a single IP address
- Repeated HTTP error responses (401, 403, 404)

---

## Features

- Server log parsing
- Detection of repeated failed logins
- Suspicious endpoint access detection
- Request rate anomaly detection
- Security event classification
- Automatic report generation
- Structured JSON output

---

## Example Log Entry

Example of a log entry that can be analyzed:

192.168.1.10 - - [07/Mar/2026:10:32:15] "GET /admin HTTP/1.1" 401


LogSentry extracts information such as:

- IP address
- timestamp
- HTTP method
- endpoint
- response status

---

## Suspicious Activity Detection

The detection engine analyzes logs and identifies patterns such as:

### Brute Force Attempts

Multiple failed login attempts from the same IP within a short period of time.

Example detection:

IP: 192.168.1.15
Type: Brute Force Attempt
Attempts: 21

### Sensitive Endpoint Scanning

Repeated access attempts to endpoints such as:

/admin
/login
/wp-admin
/config

### Error Pattern Detection

Frequent HTTP error responses such as:

401 Unauthorized
403 Forbidden
404 Not Found


These patterns may indicate vulnerability scanning or automated attacks.
